## Description (4000 characters max)
```
Essence QC vous aide à trouver le meilleur prix d'essence au Québec, en temps réel. Fini de payer trop cher — comparez les prix de toutes les stations-service du Québec sur une carte interactive ou dans une liste triée par prix ou par distance.

🗺️ CARTE INTERACTIVE
• Visualisez toutes les stations-service du Québec sur une carte Google Maps
• Prix affichés directement sur chaque marqueur, codés par couleur : vert (bon prix), orange (moyen), rouge (cher)
• Géolocalisation automatique pour voir les stations autour de vous
• Touchez un marqueur pour voir tous les détails et lancer la navigation

📋 LISTE DES PRIX
• Toutes les stations de votre zone, triées par prix ou par distance
• Économie par plein calculée pour chaque station vs la moyenne
• Avertissement intelligent : si la station est loin, l'app calcule si le déplacement en vaut la peine en tenant compte du carburant pour s'y rendre
• Navigation Google Maps en un seul toucher

🛣️ PLANIFICATEUR DE TRAJET
• Entrez votre destination et Essence QC trouve les stations les moins chères le long de votre route
• Stations recommandées selon le meilleur rapport prix/détour
• Indicateur d'autonomie pour savoir quand vous devez vous arrêter
• Ouvrez l'itinéraire dans Google Maps avec la station comme arrêt intermédiaire

⭐ MODE COSTCO
• Activez le mode Membre Costco pour voir les 3 stations Costco les plus proches en un coup d'œil
• Bannière dédiée avec prix et distance, toujours visible en haut de l'écran
• Stations Costco mises en évidence avec un style distinctif sur la carte et dans la liste

⚙️ PERSONNALISATION
• Choisissez votre type de carburant : Régulier, Super ou Diesel
• Ajustez le rayon de recherche : 5, 10, 25 ou 50 km
• Configurez la taille de votre réservoir pour des calculs d'économie précis

📊 DONNÉES FIABLES
• Toutes les données proviennent de la Régie de l'énergie du Québec
• Les prix affichés sont une obligation légale des stations-service
• Mises à jour en temps réel

Essence QC est gratuit, sans publicité et sans compte requis. L'application est développée par la communauté open source Tailed Community.
```

## App Review Notes (optional, 4000 chars)
```
Essence QC is a gas price comparison app for the province of Quebec, Canada. It displays real-time fuel prices from the Régie de l'énergie du Québec (Quebec energy regulator), which is public data.

The app does not require sign-in. It uses the device's location to show nearby gas stations. Location access is requested at launch to display relevant stations.

The app has no in-app purchases, no user accounts, and no user-generated content.

To test: Open the app, allow location access (or deny — it defaults to Montreal), and browse gas prices on the map or list view. You can tap any station for details and navigation.
```
